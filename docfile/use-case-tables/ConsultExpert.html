Use Case Description

Use Case No

UC-06

Use Case

Consult Expert

Goal

To allow a farmer to get expert advice on agricultural matters.

Preconditions

Farmer must be logged into the system.

Success End Condition

Farmer receives a consultation from an agricultural expert.

Failed End Condition

Farmer does not receive a consultation.

Primary Actors

Farmer, Agricultural Expert

Secondary Actors

System

Trigger

Farmer initiates a consultation request.

Main Success Flow

Step

Action

1

Farmer navigates to the "Consult Expert" page.

2

Farmer selects an expert from the list.

3

Farmer sends a consultation request with details of the issue.

4

The expert receives the request and provides a consultation.

5

Farmer receives the expert's advice.

Alternative Flows

Step

Branching Action

4a

If the selected expert is not available, the system notifies the farmer. Farmer can choose another expert or wait.

Quality Requirements

Step

Requirement

1

The system should facilitate real-time communication if possible.

The consultation history should be saved for future reference.
