# Functional Requirements Table

| FR ID | Name                          | Description                                                                                                                                                                             | Stakeholders                                   | Priority |
| ----- | ----------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- | -------- |
| FR-1  | Registration System           | The system shall allow farmers, experts, customers, and data operators to register with role-specific information and document verification to create secure user accounts              | Admin, Farmer, Expert, Customer, Data Operator | High     |
| FR-2  | Authentication System         | The system should authenticate users before granting access to its features and functionalities to ensure security and privacy with role-based login                                    | Admin, Farmer, Expert, Customer, Data Operator | High     |
| FR-3  | Profile Management            | The system shall allow users to view, edit, and manage their profile information with session management to maintain updated user data                                                  | Admin, Farmer, Expert, Customer, Data Operator | Medium   |
| FR-4  | Crop Recommendation System    | The system shall provide AI-driven crop recommendations based on location, season, soil conditions, and profitability analysis to help farmers make informed decisions                  | Farmer, Admin                                  | High     |
| FR-5  | Crop Planning Guideline       | The system shall provide detailed crop planning guidelines including planting schedules, care instructions, and best practices to support successful farming                            | Farmer, Expert                                 | Medium   |
| FR-6  | Disease Diagnosis System      | The system shall provide AI-powered disease diagnosis through image analysis and symptom description with treatment recommendations to help farmers identify and treat crop diseases    | Farmer, Expert                                 | High     |
| FR-7  | Expert Consultation           | The system shall facilitate farmer-expert consultation through multiple communication channels with request tracking to provide professional agricultural guidance                      | Farmer, Expert                                 | High     |
| FR-8  | Social Feed System            | The system shall provide a community platform for sharing posts, images, discussions, and agricultural knowledge to foster collaboration among users                                    | Admin, Farmer, Expert, Customer                | Medium   |
| FR-9  | Weather Forecasting           | The system shall provide current weather conditions, 7-day forecasts, and agricultural planning recommendations to help users plan farming activities                                   | Admin, Farmer, Expert, Customer                | High     |
| FR-10 | Marketplace System            | The system shall provide a marketplace for buying, selling, and renting agricultural products and equipment to facilitate agricultural commerce                                         | Farmer, Customer, Admin                        | High     |
| FR-11 | Market Price Information      | The system shall display real-time market prices, trends, and comparative pricing across different markets to help users make informed trading decisions                                | Farmer, Customer                               | Medium   |
| FR-12 | Agricultural News and Policy  | The system shall provide categorized agricultural news, policy updates, and government announcements to keep users informed about agricultural developments                             | Admin, Farmer, Expert, Customer                | Medium   |
| FR-13 | Expert Dashboard              | The system shall provide experts with consultation management, request queuing, and response tracking capabilities to efficiently manage their services                                 | Expert, Admin                                  | High     |
| FR-14 | Crop & Diagnosis Management   | The system shall allow experts to review farmer submissions, provide diagnosis, and manage crop-related consultations to deliver professional agricultural services                     | Expert, Farmer                                 | High     |
| FR-15 | Profile Verification          | The system shall enable data operators to verify and approve user profiles and submitted documents to ensure data accuracy and user authenticity                                        | Data Operator, Admin                           | High     |
| FR-16 | Field Data Collection         | The system shall allow data operators to collect, validate, and manage agricultural field data                                                                                          | Data Operator, Admin                           | High     |
| FR-17 | Crop Info Verification        | The system shall enable data operators to verify crop information, yields, and agricultural data accuracy                                                                               | Data Operator, Admin                           | High     |
| FR-18 | Social Feed Report Management | The system shall allow data operators to monitor, moderate, and manage social feed content and reports                                                                                  | Data Operator, Admin                           | Medium   |
| FR-19 | Report Analysis               | The system shall enable data operators to aggregate farmer & crop datasets, validate and analyze trends, and prepare structured analytical reports for government/administrative review | Data Operator, Admin                           | Medium   |
| FR-20 | View Report                   | The system shall enable administrators to access, review, approve, and export the analytical reports produced by data operators                                                         | Administration, Admin                          | Medium   |
